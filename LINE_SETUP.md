# 📱 LINE通知の設定（買える金ロフトが出た瞬間に届く）

「**新着 × 🟢買える × 金ロフト★**」を検知した時だけ、あなたのLINEにプッシュします。
ハズレ（発泡）や売却済では鳴りません。**無料枠（月200通）で十分**です。

LINE Notify は2025年に終了したので、**LINE Messaging API（公式アカウント）** を使います。
1回だけ、以下の設定をお願いします（5〜10分）。

---

## 手順

### 1. 公式アカウント＋Messaging APIチャネルを作る
1. https://developers.line.biz/console/ に自分のLINEでログイン
2. **プロバイダー**を新規作成（名前は何でもOK。例：`myalerts`）
3. そのプロバイダーで **「Messaging API」チャネル**を新規作成
   （チャネル名＝公式アカウント名。例：`カムイ監視`。業種などは適当でOK）

### 2. 2つの値をコピー
- **チャネルアクセストークン**：
  チャネルの「Messaging API設定」タブ →「チャネルアクセストークン（長期）」を**発行**してコピー
- **あなたのユーザーID**：
  チャネルの「チャネル基本設定」タブ下部の **「あなたのユーザーID」**（`U` から始まる文字列）をコピー

### 3. 公式アカウントを友だち追加
「Messaging API設定」タブのQRコードを、自分のLINEで読み取って**友だち追加**。
（これをしないとプッシュが届きません）

### 4. GitHubに2つの値を登録
GitHubの `kamui-tp07-finder` リポジトリで
**Settings → Secrets and variables → Actions → New repository secret** から2つ登録：

| Name | Secret |
|---|---|
| `LINE_CHANNEL_TOKEN` | ↑のチャネルアクセストークン |
| `LINE_USER_ID` | ↑のあなたのユーザーID（Uから始まる） |

> ※値をこのチャットに貼ってくれれば、こちらで `gh secret set` で登録することもできます。

---

## 完了！
以後、6時間ごとの自動スキャンで「買える金ロフト★」を見つけると、LINEに：

```
🚨買える金ロフトTP-07が出ました！

▼メルカリ｜¥30,500
KAMUI TP-07 ドライバー ガスのみ 爆音 金属バット
https://jp.mercari.com/item/m...
```

とURL付きで届きます。あとはタップして即購入するだけ。

## テスト方法
設定後、GitHubの Actions タブ →「scan」→「Run workflow」で手動実行できます。
（ただし通知は“新着の買える金ロフト”が実際に出た時だけ。テスト通知が欲しい場合は言ってください、
一時的にテスト送信するコマンドを用意します）
